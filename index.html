I'm a Computing Science graduate from University of Glasgow. This site is a collection of projects I've worked on. If you have any questions you can contact me.

Projects

Forcelayout

A Python package for visualising high-dimensional data using force directed layout algorithms. I worked on this for my final year project and dissertation at University of Glasgow.

GUTs 2018 Tanks Bot

The winning bot for a group challenge by Morgan Stanley as part of the "Do You Have the GUTs?" 2018 hackathon.

Parkour Piggies

Race your friend in a platforming quest to find mama pig! Our first 3D game, using a voxel art style. Created in 48 hours at GUDev October 2018 game jam.

Operation Wire

Play as a virus trying to infect file system but watch out for the anti-virus! Our first Unity game created in 48 hours for Global Game Jam 2018.

Tetrio

2 player asymmetric co-op game for "Do You Have the GUTs?" 2017 Verint challenge. One player places falling tetronimo blocks to help mario grab coins and reach the goal.
